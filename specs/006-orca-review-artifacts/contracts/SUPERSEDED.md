# Superseded

The contracts in this directory are superseded by
`specs/012-review-model/contracts/`. The 012 review model
collapses 006's six review types into three artifact files
(`review-spec.md`, `review-code.md`, `review-pr.md`) with
cross/self as modes within each artifact rather than separate
files.

See [012-review-model plan.md](../../012-review-model/plan.md) for
the migration rationale and
[012-review-model contracts/](../../012-review-model/contracts/) for
the current contract definitions.
